# Ontop Calculator

เว็บแอปสำหรับคำนวณเงิน Ontop

## วิธีใช้งาน

1. กรอกยอดขาย
2. กรอกเปอร์เซ็นต์ Ontop
3. กรอกโบนัสเพิ่ม (ถ้ามี)
4. กดปุ่มคำนวณ

## Deploy ขึ้น GitHub Pages

1. สร้าง Repository ใหม่ใน GitHub
2. Upload ไฟล์ทั้งหมด
3. ไปที่ Settings
4. เลือก Pages
5. Branch = main
6. Save

เว็บจะออนไลน์อัตโนมัติ
